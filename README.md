# Hand-Gesture-Controlled-Robotic-Car
The Hand Gesture Controlled Robotic Car is an embedded systems project that enables wireless vehicle navigation using natural hand movements. The system employs an accelerometer sensor to detect hand gestures, which are processed by a microcontroller and transmitted wirelessly to a receiver mounted on the robotic vehicle. Based on the received commands, the vehicle performs directional movements such as forward, backward, left, right, and stop. This project demonstrates the integration of sensors, microcontrollers, wireless communication, and motor control to develop an intuitive human–machine interaction system.
